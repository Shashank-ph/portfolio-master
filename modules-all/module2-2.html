Vulnerability Analysis – Literature Review Activity

  • Gin & Juice Shop Website Vulnerabilities

    1. Personal Addresses and other details are visible for all previous orders – GDPR concerns
    2. Coupon code visible even without registering by valid email id - 9In&JUICE5H0P
    3. Possible to check available stocks without logging in.
    4. Possible to view username and password in cleartext. (Username : carlos Password : hunter2)
    5. Unable to process / complete order as the user is flagged for deletion




    6. PCI DSS concerns: As we can login with admin account and can view order details as well as payment (Debit Card) details.

    Issues or challenges

    1. Unable to pinpoint the vulnerability and the CVE id. Example : Buffer overflow has 13 examples all belonging to different vulnerabilities and information. Solution : Search manually on NVD website (NVD - Search and Statistics) to know more about applicable vulnerabilities.

    2. NVD linkage of vulnerabilities not given on Gin & Juice website. Had to research manually. Solution : Run the search manually and with reference to other exploits available on the internet / youtube and then run those in order to find expected vulnerable results.
